## Regenerating Figures

Figures are pre-committed and do not need to be regenerated to render the executive report. To regenerate:

- **fig1–fig4**: Run `quarto render Analysis/case_study_3_RQ1_analysis.qmd` (~10 min)
- **fig5–fig6**: Execute `Analysis/CS3_RQ2_analysis.ipynb` after running the hhh4 R script (~60–120 min total)

See the root `README.md` for full step-by-step instructions.
